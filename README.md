Projeto desenvolvido na semana Spring React da Escola DEVSuperior.

O projeto foi desenvolvido em Java 17.

Front-end foi usado:
◾ HTML
◾ CSS
◾ Componentes React
◾ DatePicker
Além de estar implantado no Netlify

Back-end foi usado: 
◾ Banco de Dados
◾ DataBase seed
◾ API REST
◾ Integração com SMS via Twilio
Com implantação no Heroku
